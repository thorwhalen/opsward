opsward.cli.generate_cmd(*project_roots: str, write: bool = False, format: str = 'text')[source]

Generate missing AI setup artifacts for one or more projects.

By default, shows what would be created (dry run). Use –write to actually write files. Existing files are never overwritten.

Parameters:

  • project_roots – one or more paths to project directories

  • write – actually write files (default: dry run)

  • format – output format — ‘text’ or ‘json’

opsward.cli.install_skills_cmd(target: str = '.', *, global_install: bool = False, agents: bool = True, write: bool = False)[source]

Install opsward’s Claude Code skills (and agents) into a project or globally.

By default, shows what would be created (dry run). Use –write to actually write files. Existing files are never overwritten.

Parameters:

  • target – project directory (ignored when –global is set)

  • global_install – install into ~/.claude/ instead of the project

  • agents – also install agent definitions (default: True)

  • write – actually write files (default: dry run)
